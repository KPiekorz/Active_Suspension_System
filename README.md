## Model systemu zawieszenia adaptycyjnego
![model_picture](https://user-images.githubusercontent.com/52105679/128628600-c195fbac-2ef6-40db-b4ae-7a75dfccfc96.png)
![model_state_space](https://user-images.githubusercontent.com/52105679/128628603-87039a83-bb77-4b60-8424-d3ac68fe2741.png)

W celu stworznie modelu matemetycznego zawieszenia się, posłużono się prostymi równaniami fizycznymi, wynikającymi z II zasady dynamiki Newtona. Koło przybliżono jako masę przymocowaną do zawieszenia na sprzęznie z tłumikiem, zawieszeni jako zwykład masa nieresorowana. Samo zawieszenie skałada się róznież z spręzyny i tłumika, oraz siłownika (generuje siłę, która pozwala na zmianę charakterystyki pracy zawieszenia).
